This is a promotional video created onsite at the Fitchburg Art Museum, or FAM, by Rob Carr’s Document Design class at Fitchburg State University for the FAM Exhibit Land Ho! The show features 8 contemporary artists Carrie Crane, Sally Curcio, Leila Daw, Warner Friedman, Michele Lauriat, Sandy Litchfield, Shona Macdonald, and Sue McNally, as well as some of FAM’s Permanent collection pieces. Land Ho! will run from September 27th, 2015 until January 10th, 2016. Learn more about this exhibit Here.
